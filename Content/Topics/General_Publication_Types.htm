Publication types

tech pubs creates the following types of manuals for DC publications.

Operator's manual

Purpose: These manuals contain all the information that an operator may require to use the product, including any setup, troubleshooting, and

Scope

  • Operating instructions
  • Safety information
  • Troubleshooting?

Service manual

Purpose: Provides instructions for the service technician to service the product on site.

Scope

  • Scheduled maintenance
  • Specific maintenance tasks
  • Technical data to assist technician in performing tasks

Technical manual

Purpose: Provides the distributor and DCE's own service technicians with additional technical data and instructions to repair or completely overhaul the product

Scope

  • Theory of operation
  • Diagnostics
  • Testing
  • Adjustment and repair
  • Troubleshooting

Workshop manual

Purpose: Provides instructions to build the product in a workshop.

Scope

  • Workbench instructions
  • Major component repair and replacement
  • Component calibration
  • Jig setup
