Pronouns

Do not use pronouns, like he, she, it, and so on. The only pronoun you should use is you.

Use nouns, not pronouns

Repeat a noun instead of using a backward-referring pronoun like it, they, this, or these. Repeating the noun ensures that the reference is clear.

The Winch Assist is mounted to the base machine using standard bolts. Inspect them regularly to ensure that they are fastened securely.

The Winch Assist is mounted to the base machine using standard bolts. Inspect the mounting bolts regularly to ensure that they are fastened securely..

Rewrite for gender neutrality

Avoid pronouns like he and his in references to a person whose gender is unknown. Instead, rewrite to use the second person (you) or use plural.

To reduce operator fatigue, he should be changed over at regular intervals.

To reduce operator fatigue, the operator should be changed over at regular intervals.

See also:

Use that and which in relative clauses correctly

Very simply, only use which in a relative clause.

The winch's planetary gearboxes have inbuilt brakes which are spring applied and hydraulically released.

The cab is fitted with polycarbonate glass and a metal grille over the front window, which protects the operator from flying debris.
